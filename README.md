CUBES 2048 - NÁVOD
==================

1. CO JE TO ZA HRU?
Je to kombinace hada a hry 2048. Sbíráš kostky a zvětšuješ své číslo.

2. JAK TO SPUSTIT?
- Všechny soubory (index.html, style.css, script.js, favicon.png) měj v jedné složce.
- Dvakrát klikni na "index.html" a hra se otevře v prohlížeči.

3. OVLÁDÁNÍ
- Myš: Určuješ směr, kam kostka letí.
- Pravidla: 
  * Menší čísla můžeš sníst.
  * Větší čísla tě zabijí.
  * Stejná čísla se v tvém těle spojí (2+2=4).

4. SKÓRE
Tvoje nejlepší skóre se pamatuje i po zavření prohlížeče.
